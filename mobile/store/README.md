# App Store Metadata

This directory contains metadata and assets for app store submissions.

## Directory Structure

```
store/
├── ios/
│   ├── metadata.json          # App Store Connect metadata
│   ├── screenshots/           # iOS screenshots
│   └── previews/              # App preview videos (optional)
├── android/
│   ├── metadata.json          # Google Play Console metadata
│   ├── screenshots/           # Android screenshots
│   └── feature-graphic.png    # Feature graphic (1024x500)
└── assets/
    ├── icon-1024.png          # App icon (1024x1024)
    └── privacy-policy.md      # Privacy policy text
```
